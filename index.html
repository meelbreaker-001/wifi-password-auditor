// what it does

Core Capabilities

🔍

Password Strength Checker

Pure-Python entropy analysis — no network access required. Checks length, character variety, common-password databases, and estimates offline crack time.

📡

Handshake Analyzer

Wraps aircrack-ng and hashcat to run dictionary attacks against WPA/WPA2 handshake captures on authorized networks.

📄

PDF Report Generator

Automatically produces a formatted PDF audit report with entropy metrics, strength ratings, and actionable recommendations using reportlab.

🧪

Unit Tested

Full test suite in tests/ covering entropy math, strength classification, and edge cases. CI runs automatically on every GitHub push via GitHub Actions.

⚙️

YAML Configuration

Customize wordlist paths, report output directories, entropy thresholds, and tool locations in a single config/settings.yaml file.

🛡️

Authorization Gate

All handshake-related operations require explicit in-terminal authorization confirmation before proceeding — keeping ethical use front and center.

Installation & Usage

1

Clone the repository

git clone https://github.com/yourusername/wifi-password-auditor.git
cd wifi-password-auditor
2

Set up virtual environment & install dependencies

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3

Check password strength (safe, works anywhere)

python -m src.wifi_auditor.main --check-strength "YourP@ssword!"
4

Generate a PDF report

python -m src.wifi_auditor.main --check-strength "YourP@ssword!" --report
5

Analyze a handshake (authorized networks only)

python -m src.wifi_auditor.main \
--analyze-handshake capture.cap \
--wordlist wordlists/rockyou.txt \
--bssid AA:BB:CC:DD:EE:FF
6

Run unit tests

pip install pytest
pytest tests/ -v
